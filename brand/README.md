# GNS Brand Book

A concept brand system for the **Gwei Name Service**, a neutral, ownerless fork of wei-names.

**Version:** v0.5 · Draft · by pray.gwei

> This is an independent concept, offered to the community. GNS already has an icon in the repo (`gns-icon.svg`); this proposes a fuller system around it. Take what's useful, ignore the rest.

The full brand book is in **[GNS-Brand-Book.pdf](GNS-Brand-Book.pdf)**.

## Logo

A single green dot, the dot in `.gwei`. Wordmark: `gns●gwei`, set in Space Grotesk, with the dot as the one green accent. Short form: GNS. See [`assets/gns-dot.svg`](assets/gns-dot.svg).

## Type

- Display & labels: **Space Grotesk**
- Body: **Inter**
- Mono (CIDs, prices, contract): **IBM Plex Mono**

## Voice — "Believe, then deliver."

Simple, direct, action-oriented, jargon-free. Say less, build more. Every claim plain and provable, straight from the contract, with nothing hidden or embellished. People-driven: anyone can contribute.

## Themes (backed by code)

- **Immutable** — fees are compile-time constants; no admin, no upgrade path.
- **Unruggable** — no owner, no withdraw; the ETH is locked in the contract forever.
- **Governance-free** — no token, no DAO, no vote.

## Illustration

Flat, two-color, geometric. Green for the subject, near-black for structure and labels. See the icons in [`assets/`](assets/).

## What GNS is for

Not a replacement for ENS, but a complement. Built for the masses, the underdogs, the cypherpunks, the people who just do things, for the right reasons. Everyone can contribute, nobody is in control.
